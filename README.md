# Sistema Solar Animado

Este proyecto es una simulación visual e interactiva del sistema solar hecha con HTML, CSS y JavaScript.

## Características
- Animación de los planetas girando alrededor del sol mediante animación CSS.
- Control deslizante (esquina superior derecha) para ajustar la velocidad de rotación de las órbitas.
- Puedes mover cualquier planeta manualmente arrastrándolo con el mouse.
- Si dos planetas se tocan (colisionan), ambos explotan con una animación visual.
- Incluye un agujero negro animado, cometas, estrellas de fondo, galaxia, universo y asteroides.
- Diseño visual atractivo y responsivo.

## Uso
1. Abre el archivo `index.html` en tu navegador web.
2. Usa el control de velocidad para aumentar o disminuir la velocidad de rotación de los planetas.
3. Haz clic y arrastra cualquier planeta para moverlo manualmente.
4. Si dos planetas chocan, verás una animación de explosión y ambos desaparecerán.

## Estructura del proyecto
- `index.html`: Estructura principal del sistema solar.
- `styles.css`: Estilos y animaciones para el sistema solar, cometas, agujero negro, galaxia, universo y asteroides.
- `script.js`: Lógica para el control de velocidad, movimiento manual y colisiones.

## Personalización
Puedes modificar los colores, tamaños y velocidades editando las variables CSS en `styles.css` o los valores en `script.js`.
